## Project Overview
Residate is a premium booking and management platform features:
- **Dashboard:** Revenue tracking, pending requests, client management.
- **Client CRM:** Active status logic, invitation system.
- **Settings:** Security (2FA simulation), profile management, data export.
- **Tech Stack:** Next.js 14, TailwindCSS, Lucide Icons, LocalStorage persistence.

## Key Files & Structure
- `app/dashboard`: Main application logic.
- `lib/store.ts`: State management (Zustand) and persistence.
- `RESIDATE_REPORT.md`: Detailed report of the latest features implemented.

This directory serves as a stable copy of the project at this point in time.
